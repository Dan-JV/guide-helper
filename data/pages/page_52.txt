## Description
### Type
K ‚Äì Key, data entries in key fields are mandatory
N ‚Äì Numeric
X ‚Äì Alphanumeric, characters
% ‚Äì The field may be deleted/left blank
D ‚Äì Date field, DDMMYYYY

### File name
The file name will not be ‚Äòchecked‚Äô which means that you can decide what the name of the file should be.
However, a file name can only be loaded into the salary system once. The maximum number of characters allowed is 20 and these characters may a combination of numbers and letters.

In a csv-file, the following characters are not allowed in data; ‚Äô,‚Äô (comma) ‚Äô;‚Äô (semicolon) ‚Äù (quotation marks) as these are used as separators.¬†

The ordering of the records is of no importance. During loading, the salary system performs the sorting.
### Data check
Data check is carried out using the same rules that govern manual entries.
